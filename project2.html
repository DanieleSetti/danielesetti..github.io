Stage at Bending Spoons

On July 4, 2020, I successfully completed my second curricular internship at Bending Spoons as UI/UX designer, acquired many important skills and became a junior UI/UX designer.

Overview

Bending Spoons S.p.A. is an Italian mobile application developer , founded in 2013 and based in Milan. The company is known primarily for iOS mobile apps, including Splice, 30 Day Fitness, and Live Quiz. Bending Spoons is one of the world's leading mobile developers, by number of app downloads

During these internships, I learned many different design techniques such as: User Flow Map, Personas, Mobile UI, UX Basics, Landing Page, Personal Branding, Logo Design, Lettering, Typography Exercises and Developing my own Mobile App.

EasyCook

My final work was to create my own mobile application , I start with 3 app ideas:

  • The first idea was to create a running app, I wanted to create a running app where people could meet and run together using the app.

  • Second Idea has been created a cooking app, with recipes, but with more functions such as sorting the foods that are missing for the selected recipe, creating a shopping list, where you can also put the expiry data of the foods in your refrigerator and many other functions.

  • And the latest idea was to create a platform similar to Change.org, but for mobile devices and make it anonymous and where people could discuss a lot of different topics and create polls.

But in the end, I chose the cooking app.

Initial Process

Before starting to design of the app, I needed to collect as much information about users as possible in order to know what to focus on, which features I would use, and which ones would be ineffective and generally unnecessary for the application.

Therefore, I created a questionnaire through Google Forms and received more than fifty answers from a completely different sample of age, sex and marital status, then I analyzed the data and was able to understand what users need

We also created MVP, Personas and User Flowmap.

  • MVP (minimum viable product) design is a process for creating functional, production-ready products that serve user needs. Prototypes are not fully functional, nor are they ready for production. In some cases, minimum viable prototypes are created which are like a hybrid of the two.

  • Personas are fictional characters, which you create based upon your research in order to represent the different user types that might use your service, product, site, or brand in a similar way. Creating personas helps the designer to understand users' needs, experiences, behaviors and goals.

  • User flow is a process or path taken by a prototypical user on a website or app to complete a task. The user flow takes a user from an entry point through a set of steps towards a successful outcome and final action, such as completion of order. As a designer, you must be clear about what you're working on.

UX Process

For a designer, User experience(UX) design is the process to create products which provide meaningful, relevant and flexible experiences to users. This involves the whole design process of collecting and uniting the product, including aspects like branding, design, usability, accessibility and functionality.

When I was confident in the ideas, I started with UX design without concentrating on the visual part of the application, because in the beginning it is important that the design works and the user feels comfortable when using the application.
